How to add another demo later:
1. Make a new folder, for example: barber-demo
2. Put that demo's main file inside as: barber-demo/index.html
3. Open index.html
4. Duplicate one of the existing template cards
5. Change the text
6. Change the preview link to: barber-demo/index.html

Important:
If a demo does not open correctly, make sure its main file is named index.html.
