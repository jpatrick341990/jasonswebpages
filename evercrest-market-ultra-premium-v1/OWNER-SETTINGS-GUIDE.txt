OWNER SETTINGS GUIDE

Open site-settings.html after signing in through account.html.

What the owner can change there:
- Store name
- Tagline
- Announcement bar text
- Utility bar text
- Footer intro note
- Support email and phone
- Instagram, Facebook, Pinterest, TikTok, and YouTube links
- Store logo upload
- Theme preset, colors, fonts, spacing, card style, button style, and more

What happens when fields are blank:
- Empty social fields stay hidden
- No uploaded logo means the default image/logo path stays in place

How to save safely:
- Save brand settings first
- Then tune the theme
- Then test a few pages across the site before publishing
- Use the settings file export button before big changes so you have a backup


Homepage rows can now be turned on or off from Site Settings without touching the code.


Evercrest v1 note: the Site Settings page now includes homepage section visibility toggles, so owners can show or hide major homepage rows without editing code.
