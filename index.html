How-To's

Adding more demos later is simple if you follow the same folder pattern.

1. Duplicate one demo card in index.html.
2. Make a new folder like barber-demo or shop-demo.
3. Put that demo's main file inside as index.html.
4. Change the preview link to match the new folder name.
